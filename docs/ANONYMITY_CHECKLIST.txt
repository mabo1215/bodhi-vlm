================================================================================
TNNLS 双匿名投稿 — 匿名检查清单 (Submission Anonymity Checklist)
================================================================================
投稿前请逐项确认，避免泄露作者身份。

□ 1. 正文 (main.tex / main.pdf)
   - 作者姓名、单位、致谢、通讯邮箱已移除或置于 \begin{comment}...\end{comment}
   - 当前使用 “Author names omitted for review” 且无其他可识别信息

□ 2. 补充材料 (Supplementary)
   - 无作者姓名、单位、个人致谢
   - 无 “We thank … at [Institution]”、无可识别基金/项目编号（若需匿名）
   - 代码/数据说明中无个人仓库链接、无真实姓名

□ 3. 代码与数据
   - 若提交代码：使用匿名仓库（如 Anonymous GitHub / Zenodo 匿名链接）或说明 “Code will be released upon acceptance”
   - 正文与补充材料中不出现可识别仓库 URL（如 github.com/username/...）
   - 数据说明中不包含可识别来源（若会暴露单位/个人）

□ 4. 致谢 (Acknowledgments)
   - 正文与补充材料中致谢已删除或改为匿名表述（如 “We thank the anonymous reviewers” 仅用于修订稿，投稿时通常不出现）

□ 5. PDF 元数据 (Metadata)
   - 用 Acrobat 或其它工具检查：文件 → 属性 → 描述
   - 删除/清空：作者、单位、创建者、生产者、标题中的可识别信息
   - 若用 LaTeX 编译：在 .tex 中可加 \pdfinfo{/Author () /Creator ()} 等留空或泛化

□ 6. 其他
   - 图表内无署名、无单位 logo
   - 自引时使用 “Author et al. [X]” 或匿名引用，不写 “our previous work [Y]”
   - 若此前有 arXiv/会议版本：Cover Letter 中说明与本次投稿的差异及未双重投稿（见 Cover_letter.rtf）

================================================================================
检查完成后可打勾保存，或打印本清单随投稿材料留存。
================================================================================
